# 插件与元数据刮削

插件为 ML-Maid 提供元数据刮削能力:不必手动填写游戏资料,直接在添加/编辑表单里搜索网站(如 [VNDB](https://vndb.org)),点几下就能把标题、开发商、发售日期、评分、标签、简介、封面和背景图应用到表单。

官方插件发布在 [ML-Maid_Plugins](https://github.com/Kurosu-Ti01/ML-Maid_Plugins) 仓库。

## 安装插件

打开**设置 → 插件**,两种安装方式:

- **导入压缩包(推荐)**:点击**导入插件**,选择插件的 `.zip` 文件。ML-Maid 会自动校验并解压;导入同一插件的新版压缩包即为原地升级。
- **手动放置**:点击**打开插件目录**,把插件文件夹(内含 `manifest.json` 和脚本)放进去,然后点击**刷新**。

新装的插件立即可用,无需重启应用。

::: warning 只安装你信任的插件
插件是作者编写的 JavaScript,可以通过 ML-Maid 访问任何公开网站。请只从官方仓库或你信任的作者处获取插件;插件的 `main.js` 是可读的普通代码,想确认它做了什么可以直接打开查看。
:::

## 管理插件

插件列表的每一行显示名称、版本和类型,并提供:

- **启用开关** —— 禁用的插件保留在磁盘上,但不再出现在刮削来源列表中。标记为*不支持*的插件(为其他 ML-Maid 版本编写)无法启用。
- **删除按钮** —— 确认后删除插件文件夹,完成卸载。
- **刷新** —— 重新读取插件目录;手动改动或复制文件后使用。

## 刮削元数据

1. 打开**添加游戏**表单(或**编辑**已有游戏),输入游戏标题。
2. 点击标题输入框旁的**刮削**按钮。
3. 选择来源(如果装了多个刮削插件),调整搜索词,点**搜索**。
4. 在结果列表中点击匹配的候选。
5. 在预览界面勾选想要应用的字段。背景图可以从提供的截图里挑一张 —— 不想改就取消勾选背景。
6. 点击**应用**。字段填入表单;封面和背景下载后显示在**媒体**页,仍可继续裁剪。

在你点击表单的**保存**之前,任何内容都不会写入游戏库 —— 直接关闭表单会丢弃刮削到的图片。编辑已有游戏时,标签等列表字段会与现有内容合并,而不是整体覆盖。

::: tip
媒体页的三个**从 URL 添加**按钮使用同一套下载机制:粘贴任意图片直链即可设置图标、背景或封面,无需刮削插件。
:::
