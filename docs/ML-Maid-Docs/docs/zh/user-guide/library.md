# 管理游戏库

## 添加与编辑

点击游戏库顶部的“添加”。标题是核心字段；类型、开发商、发行商和标签支持选择已有值或创建新值。发布日期使用日期选择器，社区评分与用户评分范围为 0–100，简介按行保存为多个段落。

“高级”页包含安装路径、安装大小、累计时长、上次游玩、监控模式、进程名和转区方式。累计时长与上次游玩通常由启动监控自动更新。保存后游戏会立即出现在列表中；选择游戏并点击“编辑”可修改。删除需要二次确认，并清理图片和关联元数据。

## 图片、链接与动作

“媒体”页可设置图标、背景和封面。支持常见图片格式，也可从 EXE、DLL、ICO 或快捷方式中提取图标。文件先进入临时目录，保存时才移动至 `library/images/<uuid>/`；取消表单会清理临时文件。

“链接”页保存 VNDB、官网等 URL，详情页会用系统默认浏览器打开。“动作”页用于配置文件启动动作：填写动作名称并选择可执行文件。详情页“启动”使用第一个文件动作。

::: warning
脚本动作尚未实现；动作参数当前只保存、不执行。需要参数时可暂时封装为 `.bat` 或 `.cmd`。
:::

## 搜索、筛选与排序

搜索框按标题即时过滤。筛选支持类型、开发商、发行商和标签的组合条件；排序支持名称、添加日期、上次游玩和用户评分，并可选择升序或降序。筛选与排序会保存到设置文件，下次启动继续生效。
