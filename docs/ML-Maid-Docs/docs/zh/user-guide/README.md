# 用户文档

ML-Maid 0.5.0 是面向 Windows 的视觉小说管理器，用于集中保存游戏资料、启动游戏并统计游玩时长。

1. [开始使用](./getting-started.md)：安装、升级、数据位置和界面导航。
2. [管理游戏库](./library.md)：添加、编辑、图片、链接、搜索和筛选。
3. [启动与游玩记录](./launching.md)：启动动作、进程监控和日文转区。
4. [统计、设置与故障排查](./statistics-settings.md)：统计视图、偏好设置和常见问题。

::: warning 当前限制
“创建快捷方式”“游戏信息”“备份存档”和“脚本”仍是占位入口；文件动作的命令行参数也尚未传给启动进程。
:::
