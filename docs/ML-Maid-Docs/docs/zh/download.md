## 应该选择哪个版本？

- **NSIS 安装版：**普通个人电脑的推荐选择，按当前用户安装并附带卸载程序。
- **MSI 安装版：**适合需要 Windows Installer 部署或集中管理的环境。
- **便携版：**无需安装，解压后直接运行。程序会把数据保存在 `ML-Maid.exe` 旁，请确保目录可写。

ML-Maid 0.5.0 及之后的版本支持 Windows 10/11 x64，并使用系统 WebView2 Runtime。
