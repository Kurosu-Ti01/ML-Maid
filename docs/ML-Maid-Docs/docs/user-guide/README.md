# User Guide

ML-Maid 0.5.0 is a Windows visual novel manager for organizing game metadata, launching titles, and tracking playtime.

1. [Getting Started](./getting-started.md): installation, upgrades, data locations, and navigation.
2. [Managing the Library](./library.md): metadata, artwork, links, search, filters, and sorting.
3. [Launching and Playtime](./launching.md): launch actions, process monitoring, and Japanese locale modes.
4. [Plugins and Metadata Scraping](./plugins.md): installing plugins and filling game details automatically.
5. [Statistics, Settings, and Troubleshooting](./statistics-settings.md).

::: warning Current limitations
Create Shortcut, Game Information, Save Backup, and Script are placeholders. Command-line parameters on file actions are stored but are not passed to the process.
:::
