## Which build should I choose?

- **NSIS installer:** the best default for personal computers. It installs for the current user and includes an uninstaller.
- **MSI installer:** useful for Windows Installer-based deployment or administration.
- **Portable build:** extract and run without installation. Keep the folder writable because application data is stored beside `ML-Maid.exe`.

ML-Maid 0.5.0 and later support Windows 10/11 x64 and use the system WebView2 Runtime.
